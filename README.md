# Sql_Terra_Project

A collection of SQL scripts and resources for cleaning and preparing data used in dashboards and production workflows.

## Contents

- `Demand_Cleaning.sql` — SQL script for cleaning and transforming demand-related data.
- `Production_Data_Cleaning.sql` — SQL script containing production data cleaning and transformation logic.
- `DashBoards/` — Directory containing dashboard-related files (visualizations, exports, or dashboard SQL).
- `data-resources/` — Directory for source data samples, lookup tables, or other data assets used by the scripts.

## Purpose

These scripts are intended to provide reusable data cleaning steps that can be run in a SQL environment to prepare datasets for analysis and dashboards.

## Usage

1. Review the SQL files to understand the operations they perform before running them.
2. Make a backup or work on a copy of your dataset before applying transformations.
3. Run the scripts in an environment compatible with the SQL dialect used in the files (adjust syntax for your database if needed).

Example:

- Open `Demand_Cleaning.sql` in your SQL client and execute the statements on a test database.
- Inspect results and adjust column names or types as required by your schema.
